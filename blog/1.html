时间复杂度精简计算方式讲解：

首先是n^2的复杂度

以冒泡排序为例，冒泡排序时间复杂度是n^2，因为排序过程需要正向遍历两个数组，也就是n*(n + n - 1 + n - 2 .. + 1) == n * (1 + n) / 2 = n/2 + n^2 / 2 ≈ n^2 / 2 // 累加部分算法 :1 + 2 + 3 ... + n = (n + 1)/ 2 然后是logN复杂度

以快速排序为例，单标记快排的计算方式是，每次做二分，然后左右分别排序。换句话说，如果每次等分，第一次就是n，第二次就是2*n/2，第三次就是4*n/4 // 每次二分结束后，下一次其实是把每层的结果分别计算 也就是说，每层的计算次数可以看做n，这时候我们知道有多少层，这个层数就贴近了时间复杂度

设有x层，二分的计算方式是N = 2^0 + 2^1 + 2^2 + ... + 2^(x-1) = 2^x - 1 // 基本算法，思考一下就知道了。如果不知道看标注1

≈2^x,所以 x = log N

时间复杂度 O(N) = 单层复杂度N * 层数（二分次数）logN = NlogN
